# KFitness (PHP base API)

    Simple php base web service for KfitnessMobile

##### Requirements

* php
* xampp
* mysql
* composer

##### Restore dependencies

> composer install


##### .env file

```.env
DB_HOST="localhost:port_no"
DB_USER="db_user_bname"
DB_NAME="db_name"
DB_PASSWORD="db_password"
API_COOKIE="key"
```
